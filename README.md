# Phoenix System Bot

مشروع **Phoenix System** هو بوت ديسكورد مميز يحتوي على:
- نظام تذاكر احترافي
- أوامر إدارية عامة
- دعم للبريفكس والسلاش
- دعم للغة العربية

## كيفية الاستخدام
1. قم بإضافة البوت إلى سيرفرك.
2. استخدم الأوامر من خلال البريفكس أو سلاش كوماند.
